# Create Real-time Backend

Create database tables and enable real-time subscriptions in the connected Supabase project. This allows your application to use Supabase Realtime for instant data synchronization across connected clients.

## Features

- Create new database tables with columns and constraints
- Enable real-time subscriptions on tables (Realtime)
- Create PostgreSQL functions for backend logic
- Set up Row Level Security (RLS) policies
- Manage database schema automatically

## Usage

Use this tool to set up backend infrastructure for your application when Supabase is connected. The tool will:

1. Create tables with the specified columns and types
2. Automatically enable RLS policies
3. Configure real-time subscriptions
4. Create backend functions as needed

## Requirements

- Supabase must be connected to the project
- Valid table names and column definitions
- Proper SQL syntax for any custom functions

## Output

Returns confirmation of successful schema operations and any errors encountered during execution.
